# KAIROS AGENT

## Metodologia da Base de Conhecimento

### Objetivo

Organizar o KAIROS como memoria estendida de Christian, com capacidade de aprender contexto, preservar decisoes e recuperar conhecimento util no momento certo.

### Principios

1. Orientacoes definem como o KAIROS pensa.
2. Memorias definem o que o KAIROS sabe.
3. Documentos definem o que o KAIROS consulta.
4. Projetos definem onde o conhecimento deve ser aplicado.

### Ordem de alimentacao

1. Orientacoes
2. Memorias
3. Documentos
4. Projetos

### Regras de uso

- Orientacoes entram diretamente no prompt base.
- Memorias registram fatos persistentes, preferencias e decisoes.
- Documentos alimentam busca contextual e RAG.
- Projetos conectam conhecimento ao contexto operacional.

### Meta de maturidade

- 100 orientacoes
- 500 memorias
- 300 documentos
- 50 projetos
